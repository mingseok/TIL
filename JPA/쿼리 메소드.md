## 만약, 공통 기능이 아니라면?

예시를 보자면 이렇다.

```java
public interface MemberRepository extends JpaRepository<Member, Long> {

    List<Member> findByUsername(String username); // 이런 경우
}
```

위 코드를 보면, `Username` 이라는 것은 도메인에 특화된 경우이다.



### 결론은, 내가 따로 구현하지 않아도 `Spring-Data-Jpa`가 구현해준다.
