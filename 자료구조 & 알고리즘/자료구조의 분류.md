# 자료구조의 분류

### 자료구조

- **단순구조**
    - 정수
    - 실수
    - 문자
    - 문자열
 - 선형구조
    - 연결 리스트
        - 단순 연결리스트
        - 이중 연결리스트
        - 원형 연결리스트
    - 스택
    - 큐
    - 덱
- 비성형구조
    - 트리
        - 일반트리
        - 이진트리
    - 그래프
        - 방향그래프
        - 무방향그래프
- 파일구조
    - 순차파일
    - 색인파일
    - 직접파일
