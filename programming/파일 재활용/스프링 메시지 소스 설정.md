## 스프링 메시지 소스 설정

스프링 부트 메시지 소스 설정

스프링 부트를 사용하면 다음과 같이 메시지 소스를 설정할 수 있다.

<br/>

`application.properties`

```html
spring.messages.basename=messages,config.i18n.messages
```



MessageSource 를 스프링 빈으로 등록하지 않고, 스프링 부트와 관련된 별도의 설정을 하지 않으면 messages 라는 이름으로 기본 등록된다. 

따라서 `messages_en.properties` , `messages_ko.properties` , `messages.properties` 파일만 등록하면 자동으로 인식된다.

<br/>

## 메시지 파일 만들기

국제화 테스트를 위해서 messages_en 파일도 추가하자.

`messages.properties` :기본 값으로 사용(한글)

`messages_en.properties` : 영어 국제화 사용

<br/>

### 주의! 파일명은 massage가 아니라 messages다! 마지막 s에 주의하자

`/resources/messages.properties` 생성하기.

`messages.properties`

```
hello=안녕
hello.name=안녕 {0}
```

<br/>

`/resources/messages_en.properties` 생성하기.

`messages_en.properties`
```
hello=hello
hello.name=hello {0}
```
