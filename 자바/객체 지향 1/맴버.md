## 맴버란?

클래스 변수란?

```java
package ddd;

public class Calculator {

	static double PI = 3.14; //static 이 붙어 있으면 class 변수이다
	static int base = 0;
	int left, right; // static 이 없으므로 인스턴스의 맴버이다
	
	public void setOprands(int left, int right) { //처음에 메인에서 여기로 들어온다
		this.left = left;
		this.right = right;
		
	}
	
	public void sum() {
		// 더하기에 base의 값을 포함시켰다
		System.out.println(this.left + this.right + base);
	}
	
	public void avg() {
		// 평균치에 base의 값을 포함시켰다
		System.out.println((this.left + this.right + base) /2);
	}
	
	
	public static void main(String[] args) {
		Calculator c1 = new Calculator();
		Calculator c2 = new Calculator();
		
		c1.setOprands(10, 20);
		// 인자값 메서드로 보내기 
		c1.sum();
		// 30 출력
		
	    c1.setOprands(20, 40);
	    // 인자값 메서드로 보내기
	    c1.sum();
	    // 60 출력
	    
	    
	    // 핵심 !!!! 이렇게 쓰는 방법도 있다
	    // 클래스 변수 base의 값을 10으로 지정했다
	    Calculator.base = 10;
	    
	    // 40 출력
	    c1.sum();
	    
	    // 70 출력
	    c2.sum();
	}

}

// 출력 값.
// 30
// 60
// 70
// 10

```
