## 삼항연산자

조건 연산자

문법은 ‘조건식 ? 식1 : 식2 이런식이다.
```java
result = (x > y) ? x : y ; // 참이면 x 대입, 거짓이면 y 대입.
```

<br/>

삼항 연산자

```java
public class test {
		public static void main(String[] args) {

            int result = (3 > 4) ? 10 : 20;
            
            System.out.println(result);

            //출력값 20
	
		}
}
```

<br/>


## 삼항연산자 중첩 사용

```java
int a = 10; 
int b = 10; 

System.out.println((a > b) ? "a는 b보다 크다" : (a < b) ? "b는 a보다 크다" : "a와 b는 같다.");
```

<br/>

삼항연산자를 중첩해서 사용하면 if문을 대체할 수 있다.
