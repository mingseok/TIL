## 접근 제한자

### 접근 제한자란? 
맴버는 객체 자신만의 속성이자 특징이므로 외부에 공개하는 것이 반드시 좋은 것은 아니다.
<br/>그래서 객체의 맴버에 대한 접근을 제한할 때가 있는데, 이를 자바에서는 접근 제한자라 한다.


<br/>

## 제한자의 종류

1. public : 모든 접근 허용

2. protected : 같은 패키지(폴더)의 객체, 상속 관계의 객체 허용
3. default :  같은 패키지(폴더)의 객체 허용
4. private : 현재의 객체 안에서만 허용
